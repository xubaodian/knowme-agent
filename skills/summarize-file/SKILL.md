# summarize-file

Use this skill when the task is to digest file content and produce a compact summary.

## Instructions

1. Identify the file or document the user cares about.
2. Read the relevant material if it is available through the runtime.
3. Apply the summary style requested by the user, otherwise keep it concise.
4. If supporting references are present, use them for output formatting.
