# knowme-agent

A Hono and React application shell for an agent runtime.

## Scripts

- `npm run dev` runs the Hono API and Vite frontend locally.
- `npm run build` builds the Hono server and React frontend into `dist/`.
- `npm run typecheck` checks the TypeScript app without emitting files.
- `npm start` runs the compiled Hono server.
- `npm test` type-checks the app, builds the server, and runs tests with Node's built-in test runner.

## Local URLs

- Frontend dev server: `http://127.0.0.1:5173`
- Hono API server: `http://127.0.0.1:3000`
